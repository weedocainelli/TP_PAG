#Guia de rocket league

##Descripcion
Este proyecto es una pagina creada con html que funciona como guia para jugadores de Rocket League. Fui separando las distintas etapas del proyecto en secciones para ir teniendo una mejor organizacion. Esta incluye cosas como: informacion base del juego, mecanicas base, sistema de rangos y diferentes tips para mejorar. Tambien tiene un sistema de contacto al final para cualquier consulta/sugerencia

##Contenido 
-Menu principal con nav
-Informacion general sobre el juego
-Mecanicas basicas y video tutorial
-Tabla de rangos
-Consejos para mejorar
-Formulario de contacto
